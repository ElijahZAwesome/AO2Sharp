BanIp(String, Nullable<TimeSpan>, IClient)

Ban this player's IP address, which is easily changing.

Declaration
void BanIp(string reason, TimeSpan? expireDate, IClient banner)
Parameters
Type Name Description
System.String reason

Why the client is being banned
System.Nullable<System.TimeSpan> expireDate

When their ban will expire (null for an indefinite ban)
IClient banner

Who banned this client (null if the server banned them)
Remarks

When trying to ban a player, this should almost always be accompanied by BanHwid, otherwise a player can simply change IPs using a VPN, Proxy, or a reset of their router to join the server again. However, if using this as a method of rate limiting, banning only their IP is acceptable.
